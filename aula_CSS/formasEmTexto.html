shaped-outside

esta propriedade permite que voce defina uma forna para que o texto se envolva e flua ao seu redor

junto a esta propriedade temos algumas formas basicas

o elemento que vamos aplicar a forma com a propriedade shaped-outside precisa ter a propriedade float

  • inset()
  • circle()
  • ellipse()
  • polygon()

função inset()

pode ser usado para criar um retangulo ou quadrado com um deslocamento opcional para o texto que o envolver
